\subsection*{Summary}

\begin{remark*}[Section summary]
The compactness preview records three related ideas.

\begin{itemize}
\item A set of propositional formulas is satisfiable when one truth assignment
makes all of its formulas true.
\item A set is finitely satisfiable when every finite subset is satisfiable.
\item Propositional compactness says that finite satisfiability is enough for
satisfiability.
\end{itemize}
\end{remark*}

\begin{remark*}[Conceptual takeaway]
In propositional logic, an inconsistency always has a finite witness. If no
finite witness exists, then the whole set of formulas can be satisfied.
\end{remark*}
