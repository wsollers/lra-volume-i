# Propositional Logic Rebuild

The previous active propositional-logic chapter has been archived under
`archive/pre-2026-governance-rebuild/`.

The active chapter is now a skeleton only. No mathematical content has been
rewritten yet, and no theorem, proof, exercise, or capstone content has been
regenerated.

Section directories use semantic names, not numeric prefixes. Render order is
controlled by `notes/index.tex` and `proofs/index.tex`, not by filesystem
ordering.

Proofs are topic-grouped under `proofs/<topic>/` while still rendering through
`proofs/index.tex`.
