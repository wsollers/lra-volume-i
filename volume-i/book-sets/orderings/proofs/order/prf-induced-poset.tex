\begin{proposition*}[$\leq_f$ is a Partial Order iff $f$ is Injective]
For a partially ordered set $(B,\leq')$ and a function $f:A\to B$, the induced
relation $\leq_f$ is a partial order on $A$ if and only if $f$ is injective.
\end{proposition*}
